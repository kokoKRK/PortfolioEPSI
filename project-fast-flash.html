Fast Flash

Projet personnel : site vitrine et e-commerce pour une marque fictive de cocktails premium en bouteille, avec livraison rapide à Montpellier — catalogue dynamique, panier, offres et espace client.

Voir le site

Contexte & objectifs

Fast Flash est un projet personnel imaginé autour d'un concept de livraison de cocktails artisanaux en bouteille à Montpellier. L'idée : une expérience premium, rapide et accessible, pensée pour les étudiants et jeunes actifs.

  • Créer une identité de marque cohérente (visuels IA, ton premium).
  • Présenter un catalogue de cocktails avec fiches détaillées.
  • Mettre en place un parcours d'achat (panier, offres, compte).
  • Optimiser le référencement (meta, Open Graph, schema.org).

Stack & architecture

  • Front : HTML, CSS, JavaScript
  • Données catalogue : JSON (cocktails, commandes)
  • API : PHP (auth JWT, utilisateurs, cocktails)
  • Base : MySQL (schéma prévu)
  • Modules JS : catalogue, panier, auth, offres

Fonctionnalités principales

  • Page d'accueil avec hero, histoire de marque et sections cocktails/offres.
  • Carrousel de cocktails chargé dynamiquement depuis JSON.
  • Modales détail (ingrédients, description, ajout au panier).
  • Panier persistant et offres spéciales (Five, Duo, Pack groupe).
  • Connexion / inscription et menu utilisateur.
  • Footer structuré avec données locales (Montpellier, contact, Instagram).

Compétences mobilisées

  • Intégration front responsive et animations CSS.
  • Manipulation DOM et logique JavaScript modulaire.
  • Consommation d'API REST et gestion d'état client (panier, session).
  • SEO technique (balises meta, Open Graph, données structurées).
  • Conception produit : branding, UX e-commerce et storytelling.
