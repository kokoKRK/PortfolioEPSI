Hazmat Airsoft

Site vitrine personnel dédié aux répliques airsoft inspirées du jeu Rust : présentation des armes, univers visuel immersif et parcours de découverte. Le projet est volontairement statique : pas de connexion utilisateur ni de catalogue dynamique, faute de capacité backend disponible sur Supabase (autres projets actifs en priorité).

Contexte & objectifs

Hazmat Airsoft met en avant un catalogue de répliques airsoft calquées sur les armes du jeu Rust (AK-47, Spas-12, LR-300, Python, Sniper Bolt, etc.). L'objectif est de créer une vitrine forte, orientée image de marque et immersion, sans dépendre d'un backend actif.

  • Construire une identité visuelle brute, fidèle à l'univers Rust / survival.
  • Présenter les répliques par catégories (shotguns, grosses armes, petites armes, snipers).
  • Mettre en avant le réalisme et le lien direct avec le jeu (avis, descriptions, visuels).
  • Proposer un parcours contact pour les demandes (devis, renseignements).
  • Préparer une base front prête pour une future boutique (auth / panier non activés).

Stack & périmètre actuel

  • Front : HTML, CSS, JavaScript
  • Contenu produits : catalogue statique (images + fiches vitrine)
  • Filtrage par catégorie côté client
  • Formulaires contact / newsletter (front uniquement)
  • Hors périmètre actif : Supabase, comptes utilisateurs, panier, Stripe

Ce que le site propose

  • Hero vidéo et sections catégories / répliques / avis / à propos.
  • Galerie des répliques Rust avec filtrage par type d'arme.
  • Descriptions orientées jeu Rust et expérience terrain airsoft.
  • Section témoignages et bloc contact.
  • Design responsive, ambiance sombre et industrielle.

Ce qui n'est pas en ligne

  • Pas de connexion / inscription utilisateur.
  • Pas d'affichage dynamique des produits depuis une base de données.
  • Pas de panier ni de tunnel de paiement Stripe.
  • Pas de page profil ni d'historique de commandes.
  • Backend Supabase réservé à d'autres projets pour l'instant.

Compétences mobilisées

  • Intégration front-end responsive avec identité visuelle marquée.
  • JavaScript pour filtres, navigation et interactions UI.
  • Conception UX d'une vitrine produit immersive.
  • Structuration d'un projet web modulaire (évolution e-commerce possible).
  • Direction artistique alignée sur une licence / univers de jeu (Rust).
