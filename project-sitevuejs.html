Site Vue.js

Projet web realise avec Vue 3 pour pratiquer les composants, la navigation d'une SPA, les formulaires valides et les appels API cote front-end.

Contexte & objectifs

L'objectif de ce projet est de consolider les bases du framework Vue.js en realisant une application front interactive, orientee composants et manipulations de donnees API.

  • Construire une navigation simple entre plusieurs sections.
  • Creer un formulaire avec validation client.
  • Consommer des APIs externes et afficher les reponses.
  • Mettre en place un composant bonus de type accordeon.
  • Structurer proprement le code en composants reutilisables.

Stack & architecture

  • Framework: Vue 3
  • Bundlers/outillage: Vue CLI + Vite
  • Langages: HTML, CSS, JavaScript
  • Pattern: composants + etat local (ref, reactive)
  • API: fetch vers JSONPlaceholder et Chuck Norris API

Fonctionnalites principales

  • Navigation interne entre pages (home, contact, api, bonus).
  • Formulaire de contact avec validations et messages d'etat.
  • Testeur d'API configurable (URL, methode, body JSON).
  • Accordeon avec tableau d'utilisateurs et blague API.
  • Affichage formate des reponses JSON.

Competences mobilisees

  • Developpement front-end avec framework JavaScript moderne.
  • Conception de composants reutilisables.
  • Validation de formulaires et gestion d'erreurs.
  • Integration et traitement de flux API.
  • Organisation de projet npm (scripts dev/build/preview).
