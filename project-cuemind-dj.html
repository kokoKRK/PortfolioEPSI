CueMind DJ

Application web pour DJs : bibliothèque musicale, génération de sets par IA, mode back-to-back (B2B) assisté, playlists par style et préparation de transitions intelligentes.

Vision du projet

CueMind DJ aide les DJs à préparer leurs sets plus vite : import de morceaux, analyse BPM/clé/style, génération de playlists et suggestions de enchaînements avec explications (jeux de mots, hooks communs, etc.).

  • Importer et organiser une bibliothèque locale (MP3/WAV).
  • Générer des sets via prompt IA avec filtres (BPM, style, durée).
  • Mode B2B : l'IA propose 3 morceaux distincts avec note de transition.
  • Playlists par style, tri Serato-like (clé, BPM).
  • Profil DJ type portfolio (roadmap) : bio, remix, contact.

Stack & architecture

  • Next.js (App Router) + TypeScript
  • React, Tailwind CSS, composants UI
  • API Routes : upload, sets, B2B, classification IA
  • Analyse audio & métadonnées (BPM, style, clé)
  • Intégrations prévues : Spotify / Deezer

Avancement actuel

  • Parcours import, bibliothèque, playlists par style et découverte en ligne.
  • Générateur de set IA avec filtres et résultats structurés.
  • Panel B2B avec suggestions et notes de transition contextualisées.
  • Prochaines étapes : profil DJ public, liens playlists externes, tri avancé type Serato.
